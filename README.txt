
Cara Menggunakan File Ini:

Agar file UTS_Angka dan UTS_Huruf dapat dijalankan dengan benar, kamu harus memasukkannya terlebih dahulu ke dalam project Java berbasis Ant di NetBeans. Berikut langkah-langkahnya:

1. Buka NetBeans.
2. Buat Project Baru dengan jenis "Java with Ant > Java Application".
3. Beri nama project, misalnya: Tugas_UTS.
4. Setelah project dibuat, salin keempat file yang ada dalam  berikut ke dalam folder src/tugas_uts/ di dalam project:
   - UTS_Angka.form
   - UTS_Angka.java
   - UTS_Huruf.form
   - UTS_Huruf.java
5. Buka project di NetBeans dan pastikan GUI muncul dengan benar.
6. Jalankan file UTS_Angka.java atau UTS_Huruf.java sesuai kebutuhan.
